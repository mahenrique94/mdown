Como usar o editor ?

  • Primeiro faça o download do projeto clicando aqui.
  • Depois vá até onde o download foi feito e extraia o conteúdo do .zip.
  • Renomeie a pasta extraída de mdown-master para apenas mdown.
  • Agora copie ela e cole dentro do seu projeto.
  • Com os arquivos disponíveis dentro do projeto, o que precisamos fazer é importar o .css e .js:
  •                         
                            <link href="PASTA_EXTRAIDA/mdown.css" rel="stylesheet" type="text/css">
                            <script src="PASTA_EXTRAIDA/mdown.js" type="text/javascript"></script>
  • Para começar a usar o editor adicione a classe md-editor__init em um textarea.
  • Para aprender as marcações do editor veja a documentação.
